# Food Delivery Data Analysis

This project analyzes food delivery data using Python, Pandas, and SQL.

## Files
- orders.csv: Order-level transaction data
- users.json: User details including city and membership
- restaurants.sql: Restaurant data with cuisine and ratings
- Food_Delivery_Analysis.ipynb: Main analysis notebook

## Tools Used
- Python
- Pandas
- SQLite
- Jupyter Notebook

## Objective
To analyze revenue, user behavior, and restaurant performance for a food delivery platform.
